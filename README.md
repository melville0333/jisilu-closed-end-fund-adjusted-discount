# 集思录 封闭股基(股票型) 修正年化折价率增强脚本

在集思录封闭式基金（股票型）页面增加「修正年化折价率」列  
计算公式：(折价率 - 1.5) ÷ 剩余年限  
**值越小（越负）越好**，支持点击表头排序

## 安装方式

1. 安装 Tampermonkey / Violentmonkey / ScriptCat 等用户脚本管理器
2. 点击下面链接直接安装（推荐）：

   [![安装脚本](https://github.com/melville0333/jisilu-closed-end-fund-adjusted-discount/raw/refs/heads/main/jisilu-cf-adjusted.user.js)

或者去 Greasy Fork 安装（有安装量统计）：  
https://greasyfork.org/zh-CN/scripts/563579-%E9%9B%86%E6%80%9D%E5%BD%95-%E5%B0%81%E9%97%AD%E8%82%A1%E5%9F%BA-%E8%82%A1%E7%A5%A8%E5%9E%8B-%E4%BF%AE%E6%AD%A3%E5%B9%B4%E5%8C%96%E6%8A%98%E4%BB%B7%E7%8E%87

## 功能

- 新增「修正年化折价率」列
- 支持升序/降序点击排序（负值优先）
- 自动给前5、10、15...名标红排名
- N/A 项不会干扰排序
